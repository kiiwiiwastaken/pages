Welcome!

Welcome to Kiiwii's list of PSP homebrew! This is a collection of homebrew that I find that I use on my PSP (or just homebrew that's hilariously bad). The only exception I have with adding homebrew here is if they are still getitng updates, unless the creator of said homebrew wants it to be on this list. All of the downloads can be found here. This site is to make it easier as a one click download solution.

NOTE!

All of the following homebrew works on Inifity Pro-C with firmware 6.61. I cannot confirm that the following homebrew works on any other firmware!



Games

  • PSP Doom v1.4: It's Doom, for the PSP! It comes with the shareware wad of Doom 1 so you just need to put it on your PSP and you're ready to play Doom!


Emulators

  • MasterBoy: This is a Gameboy, Gameboy Color, Sega Master System and Sega Game Gear emulator.
  • NesterJ: This emulates the Nintendo Entertainment System/Famicom
  • PicoDrive: This emulates the Sega Genesis and Sega CD
  • TempGBA: This emulates the Gameboy Advance
  • s9xTYLme_mod: This emulates the Super Nintendo Entertainment System/Super Famicom


Utilities

  • PSP Filer: A file manager
  • UMDKiller: Dumps UMD Games to a .iso file


I have vsh plugins as well but I suggest just going into the repository to get those (which I have linked above.)
