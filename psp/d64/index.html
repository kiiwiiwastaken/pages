Daedalus X64 Compatibility List

Here is a list of n64 games that I have tested and how well it runs, if at all. This page is broken off into sections based off of versions of Daedalus X64, starting at 1.1.9 since that is the latest version as of writing (September 2020)


Here are the following settings for the emulator that I run for every games unless otherwise said.


Rom Prefrences:

  • Texture Update Check -- Disabled
  • Frameskip -- Disabled
  • Zoom -- 100%
  • Limit Framerate -- Full Speed
  • Dynamic Recompilation -- Enabled
  • High Level Emulation -- Enabled
  • Audio -- Disabled
  • Controller -- Default

Advanced Settings:

  • Dynarec Memory Optimisation -- Enabled
  • Dynarec Loop Optimisation -- Enabled
  • Dynarec Doubles Optimisation -- Enabled
  • Clean Scene -- Disabled
  • Clear N64 Depth Buffer -- Disabled
  • Double Display Lists -- Enabled
  • Video Rate Match -- No
  • Audio Rate Match -- No
  • Fog Emulation -- Disabled

Global Settings:

  • Display Info -- None
  • Viewport Size -- 4:3 unscaled (320x240)
  • Force Linear Filter -- No
  • Controller add-on -- MemPak
  • Stick Deadzone -- 28/100
  • ROM Buffering Mode -- File Cache
  • Low Battery Warning -- No
  • GUI Color -- Black
  • Language -- English

Now with all off that out of the way, here's my personal compatibility list of N64 games running on Daedalus X64!
